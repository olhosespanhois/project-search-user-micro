Documentação

Para instalar o projeto "project-search-user-micro" em sua máquina, siga as instruções abaixo:

Requisitos:

Node.js e npm (gerenciador de pacotes) instalados. Você pode baixá-los em https://nodejs.org/.

Git deve estar instalado em sua máquina. Você pode baixá-lo em https://git-scm.com/.

Abra o terminal ou prompt de comando em seu computador e execute o seguinte comando para clonar o repositório:

git clone https://github.com/olhosespanhois/project-search-user-micro.git

Navegue para a pasta do projeto que acabou de ser clonada:

cd project-search-user-micro

Instalar o Angular CLI:

Abra o terminal ou prompt de comando em seu computador e execute o seguinte comando para instalar o Angular CLI globalmente em sua máquina:

npm install -g @angular/cli

Após a instalação, você pode verificar se o Angular CLI foi instalado corretamente executando o seguinte comando:

ng version

Instalar as dependências:

Execute o seguinte comando para instalar as dependências do projeto:

npm install

Com todas as dependências instaladas e as variáveis de ambiente configuradas, você pode executar o projeto com o seguinte comando:

ng serve @station/home

Com o projeto em execução, você pode acessar a aplicação em seu navegador em http://localhost:4200 (ou a porta especificada, se for diferente).

Agora, o projeto "project-search-user-micro" deve estar instalado e funcionando em sua máquina. Em caso de dúvidas acesse o README.md ou entre em contato pelo samiranbittar@outlook.com.
